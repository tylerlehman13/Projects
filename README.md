# Tyler Lehman — Data Science Projects

A collection of sports analytics and data science projects built with Python, machine learning, and statistical modeling.

## Projects

### **<a href="./2024_NBA_Playoff_Prediction">2024 NBA Playoff Prediction</a>**
Predicted the 2024 NBA Playoffs through Python and OLS regression with a custom weighted function to determine length of series, and winner.

**Files:** `2024_NBAPlayoffs_Prediction_slides.pdf`, `2024_NBA_Bracket_Prediction.pdf`, `2024_NBA_Playoff_Code.pdf`

### **<a href="./2025-26_PistonsWinPrediction.ipynb">2025-26 Detroit Pistons Win Prediction</a>**
Predicted the Pistons' win total for the 2025–26 NBA season using Python, the `nba_api` library, and machine learning techniques.

**Files:** `2025-26_PistonsWinPrediction.ipynb`

---

### **<a href="./Classifying_Surfaces">Classifying Surfaces — Image Classification</a>**
In Python, I used SVC and Decision Tree models to classify images from a large dataset containing images of concrete. There are 3 surface types in the data: Decks, Walls, and Pavements, and each image is also labeled as Cracked or Non-Cracked

**Files:** `CMSE381_Project.pdf`, `CMSE381_ProjectSlides.pdf`

---

### **<a href="./Does Defense Win Championships In The NBA.pdf">Does Defense Win Championships In The NBA?</a>**
With Python and historical NBA statistics, I analyzed if top defenses usually ending winning the championship, inspired by the common phrase: "Defense wins championships".

**Files:** `Does Defense Win Championships In The NBA.pdf`

---

### **<a href="./NBA_ShotChart_Trends">NBA Shot Chart Trends</a>**
Analyzed shot chart data from the NBA from 2013-14 to 2025-26 through Python, `nba-api`, and machine learning techniques. Also looked into shot selection and FG% across quarters in a game, and what led to changes shot charts over seasons.

**Files:** `NBA_ShotChart.ipynb`, `NBA_request.ipynb`, `final_presentation.pdf`
